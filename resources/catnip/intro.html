Catnip

Cheat sheet:

Ctrl-Alt-O
Open a file
Ctrl-Alt-F
Create a new file
Ctrl-Alt-S
Save and compile
Ctrl-Alt-,
Save and run tests
Ctrl-Alt-R
Jump between editor and REPL
Ctrl-Alt-E
Evaluate the sexp before the cursor
Ctrl-Alt-B
Switch to an open buffer
Ctrl-Alt-I
Insert a snippet
Ctrl-Alt-1
Toggle browser frame
Ctrl-Alt-2
Toggle presentation mode
